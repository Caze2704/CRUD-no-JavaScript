# 📋 Cadastro de Usuários (CRUD com JavaScript)
## (ainda em desenvolvimento.)

Este projeto é um sistema simples de **cadastro de usuários** desenvolvido com **HTML, CSS e JavaScript puro**.
Os dados são armazenados utilizando **LocalStorage**, permitindo que as informações permaneçam salvas mesmo após atualizar a página.

O objetivo deste projeto foi praticar conceitos importantes de **JavaScript para manipulação do DOM, validação de dados e armazenamento no navegador**.

---

# 🚀 Funcionalidades

* Cadastro de usuários
* Validação de campos
* Mensagens de erro e sucesso
* Armazenamento dos dados no **LocalStorage**
* Estrutura de **CRUD (Create)** em JavaScript

---

# 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage API

# 📚 Conceitos praticados

Durante o desenvolvimento deste projeto foram praticados:

* Validação de dados

* Arrays de objetos
